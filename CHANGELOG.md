# Changelog



### [1.2.1] - 2025-12-03

#### Added

- 新增【聚焦模式】：挂前台3分钟后屏幕整体变黑色，只显示专注事项的标题和时间，右下角放一个退出按钮。倒计时结束自动取消聚焦模式。

#### Fixed

- 每日0点自动重置【专注】模块的正计时和倒计时。

#### Changed

- 修改【专注】模块的倒计时结束音效，音频已经放在 assets/audio/alarm0.mp3
- 修改了软件图标。



### [1.1.1] - 2025-12-02

#### Added

- 可导入导出数据。

#### Fixed

- 修复了退出程序时所有数据全部丢失的小BUG。

#### Changed

- 因为不会改 Bug，为了保证数据持久化，切后台后无法继续计时了。



### [1.1.0] - 2025-12-02
#### Added
- 清单模块中截止时间的颜色默认绿色，超时后才变成红色。
- 增加前台服务机制，切后台依然可继续倒计时。
- 新增【周期】模块：每天几时几分/每周第几天/每月第几天/每年几月几日定期提醒某事件，比如每月提醒机场的月卡该更新了，比如每周五充电宝该充电了。

#### Changed
- 上导航栏美化，加入艺术字，更改底色为主题色。
